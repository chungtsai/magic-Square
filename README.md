# 魔術方框：數字神祕方陣遊戲 (Magic Square Grid)

這是一個以 HTML5、CSS3 與純 JavaScript 打造的極致美觀益智遊戲。遊戲設計巧妙融合了古代傳奇「洛書九宮圖」的數學理論與現代流暢的 Web 視覺美學。

## 🌟 遊戲特色

1. **多重主題風格**
   - 🌌 **幻境星空 (Cosmic Space)**: 寧靜神祕的深色太空格調，配有緩緩流動的星空粒子與紫靛色霓虹輝光。
   - ⚡ **霓虹賽博 (Cyberpunk)**: 濃郁的夜行極簡賽博霓虹，明亮搶眼的桃紅、粉青線條，極具科幻未來感。
   - 🎮 **復古經典 (Retro Arcade)**: 懷舊感十足的街機像素橘黃暖光氛圍。
2. **多維度矩陣選擇**
   - 3x3 方陣（數字 1-9，目標常數為 15）
   - 4x4 方陣（數字 1-16，目標常數為 34）
   - 5x5 方陣（數字 1-25，目標常數為 65）
3. **雙重遊玩模式**
   - 📝 **填數模式 (Fill Mode)**: 從下方的數字銀行中點擊選取或拖曳數字填入方格中，按住格子雙擊即可清除。
   - 🔄 **置換模式 (Swap Mode)**: 方陣中已預先填滿所有被打亂的數字，你可以任意點選兩個非提示格子，將其內容直接交換。
4. **即時動態數學運算指示器**
   - 行、列與兩條對角線的邊緣都設有懸浮「總和泡泡」。
   - 每當玩家操作時，系統自動即時計算總和並顯示狀態：
     - **綠色發光泡泡**: 代表該排總和已正確契合魔術常數。
     - **紅色警告泡泡**: 代表該排已填滿，但總和不等於魔術常數。
     - **紫色虛線泡泡**: 代表該排尚有空缺。
5. **卓越的音效反饋與勝利慶祝**
   - 利用 Web Audio API 純代碼合成各類流暢短音（填數、置換、擦除、出錯、提示音與大調和弦胜利樂章）。
   - Canvas 物理模擬彩色碎紙屑 (Confetti) 動態飛落，配合精美的勝利看板與步數、計時統計。

## 📁 檔案結構

* `index.html` - 主遊戲版面與結構設計。
* `style.css` - 含 Glassmorphism（磨砂玻璃）、星空動態、各色主題設定與響應式佈局。
* `script.js` - 魔術方陣產生算法、拖曳與點擊操作引擎、計時器、音效合成與獲勝驗證邏輯。
* `confetti.js` - 獨立的 Canvas 粒子碎紙屑動畫渲染引擎。

## 🧠 遊戲玩法引導

1. 目標是讓橫排、直列、對角線的所有數字加總皆等於 **魔術常數**（例如 3x3 的常數是 15）。
2. 在填數模式下，先點擊底部的可用數字，再點擊格子即可填入；雙擊格子可以收回數字。亦可以直接將數字從下方拖曳進格子裡。
3. 在置換模式下，直接點擊一個數字，再點擊另一個數字即可對調位置。
4. 如果遇到瓶頸，可以點選「提示一個字」或「顯示解答」進行參考！
